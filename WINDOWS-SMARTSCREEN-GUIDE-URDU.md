# PaintPulse انسٹالیشن گائیڈ - Windows سیکیورٹی وارننگ

## یہ وارننگ کیوں آتی ہے؟

جب آپ PaintPulse انسٹال کرنا چاہیں گے تو Windows کی طرف سے یہ وارننگ آئے گی:

> **"Windows نے آپ کے PC کی حفاظت کی"**  
> **Publisher: Unknown publisher**

یہ **نارمل** اور **محفوظ** ہے۔ اس کی وجہ:

- PaintPulse کے پاس Digital Certificate نہیں ہے
- یہ سرٹیفکیٹ سالانہ $200-400 خرچ ہوتا ہے
- Windows ہر بغیر certificate والے app کے لیے یہ وارننگ دکھاتا ہے
- **آپ کا app مکمل طور پر محفوظ ہے**

## ✅ انسٹالیشن کا طریقہ

### قدم 1: Setup File چلائیں
`PaintPulse-Setup-0.1.7.exe` پر ڈبل کلک کریں

### قدم 2: وارننگ اسکرین
نیلی وارننگ اسکرین پر:

1. **"More info"** پر کلک کریں (اوپر کی طرف)
2. **"Run anyway"** بٹن نظر آئے گا
3. **"Run anyway"** پر کلک کریں

### قدم 3: انسٹالیشن مکمل کریں
- اب installer نارمل طریقے سے چل جائے گا
- انسٹالیشن لوکیشن منتخب کریں
- انسٹالیشن مکمل ہونے کا انتظار کریں
- Desktop shortcut یا Start Menu سے PaintPulse چلائیں

## 🔒 کیا یہ محفوظ ہے؟

**جی ہاں!** یہ وارننگ اس لیے آتی ہے کیونکہ:
- App کے پاس مہنگا code-signing certificate نہیں ہے
- Windows ابھی AMPaints کو نہیں جانتا
- یہ ایک **سیکیورٹی فیچر** ہے، وائرس کی وارننگ نہیں

**App مکمل طور پر محفوظ ہے:**
- قابل اعتماد ٹیکنالوجیز سے بنایا گیا (Electron, React, SQLite)
- کوئی بیرونی server سے رابطہ نہیں
- تمام ڈیٹا آپ کے computer پر محفوظ
- Open source بلڈ پراسیس

## خلاصہ

- ✅ SmartScreen وارننگ **نارمل اور محفوظ** ہے
- ✅ "More info" → "Run anyway" سے bypass کر سکتے ہیں
- ✅ Code signing **اختیاری** ہے اور سالانہ $200-400 خرچ
